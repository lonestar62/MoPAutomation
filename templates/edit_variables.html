Variable Guidelines
Required Fields:
  • id - Unique MOP identifier
  • title - Human-readable title
  • category - MOP category for playbook mapping
  • region - Target region
Available Categories:
  • patch-linux - Linux patching operations
  • agent-upgrade - Agent upgrade procedures
  • pipeline-only - DevOps pipeline triggers
  • git-ops - Git-based operations
  • infrastructure - Infrastructure changes
Example:

title: "Patch Critical Servers"
category: patch-linux
region: eastus2
patch_group: critical
maintenance_window: "02:00-04:00"
